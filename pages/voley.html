DEPORTE VÓLEY

El voleibol es un deporte de equipo donde dos grupos de seis jugadores intentan pasar el balón por encima de una red y hacerlo caer en el campo contrario. Se caracteriza por un máximo de tres toques por equipo para devolver el balón.

REGLAS ∧
  • Cada equipo juega con 6 jugadores en la cancha
  • El objetivo es que el balón toque el suelo del campo rival
  • Cada equipo solo puede dar un máximo de 3 toques para devolver el balón
  • No se puede retener, empujar o lanzar el balón (doble golpe o retención)
  • Un mismo jugador no puede tocar el balón dos veces consecutivas (excepto después de un bloqueo)
  • Puntuación: Se usa el sistema de puntos por jugada (Rally Point), el primer equipo en alcanzar 25 puntos con una ventaja de dos gana el set
  • El partido se gana al mejor de 5 sets (4 sets a 25 puntos y el 5º set a 15 puntos)
  • Rotación: Los jugadores deben rotar una posición en sentido horario después de que el equipo recupera el saque
TÁCTICA ∨
  • Complejo I (Recepción-Ataque): Se enfoca en la recepción del saque y la organización inmediata del ataque
  • Complejo II (Defensa-Contraataque): Se enfoca en la defensa de un ataque rival y la posterior transición al ataque
  • Bloqueo: Barrera defensiva sobre la red para impedir el ataque del rival o suavizarlo
  • Sistema 5-1: Un armador principal (colocador) y cinco rematadores (atacantes)
  • Sistema 4-2: Cuatro rematadores y dos armadores
  • Zonas de ataque: Utilizar las zonas 2, 3 y 4 para remates y la zona 6 para defensa central
ELEMENTOS BÁSICOS ∨
  • Balón de voleibol: Esférico, con cámara interior y de cuero flexible o sintético
  • Cancha rectangular: 18m de largo x 9m de ancho, dividida por una red
  • Red: Altura de 2.43m para hombres y 2.24m para mujeres (categoría adulta)
  • Antenas: Colocadas en los límites laterales de la red para demarcar el espacio de paso del balón
  • Línea de ataque: Marcada a 3m de la red; los zagueros no pueden rematar delante de ella
  • Calzado deportivo: Zapatillas con buena amortiguación y soporte de tobillo para saltos
TÉCNICAS ∨
  • Toque de dedos (Colocación): Usado principalmente por el armador para preparar el remate
  • Golpe de antebrazos (Recepción/Defensa): Usado para recibir el saque o defender ataques bajos
  • Saque: Puesta en juego del balón; puede ser de abajo, de tenis, flotante o potente (jump service)
  • Remate (Ataque): Salto y golpe fuerte con la palma para dirigir el balón al suelo rival
  • Bloqueo: Salto cerca de la red con manos rígidas para interceptar el ataque
  • Defensa de campo: Uso de pases bajos o "planchas" para evitar que el balón toque el suelo
